  one path-expansion contract even though they share one runtime payload.
- Treat local `--plugin-dir` acceptance as runtime proof only. It is not proof
  of Cursor Team Marketplace import, installation, update, or rollback.
- GitHub Actions remain intentionally unused; release blockers run locally in
  the coding session and git hooks.
