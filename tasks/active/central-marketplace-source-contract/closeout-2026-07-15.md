## Outcome

The Apple Mail repository is now a source-of-truth producer for the shared
Agentic Assets marketplace. It describes one allowlisted installable payload,
binds future releases to deterministic digests, and leaves admission,
catalog upserts, attestations, and rollback under the central marketplace's
authority.

The primary Agentic Assets selector is `apple-mail@agentic-assets`. The local
root catalogs remain an explicit standalone compatibility lane under
`apple-mail@apple-mail-mcp`; they do not claim ownership of the shared
marketplace identifier.

## Implemented controls

- Added `tools/marketplace_identity.json` as the source-side identity contract.
- Added `distribution/marketplace-payload.json` and deterministic inventory
  validation for the complete installable payload.
- Excluded source-only documentation and development inputs, including skill
  authoring READMEs, while failing on unclassified files and symlinks.
- Added bounded inspection of decompressed wheel members for unsafe paths,
  archive bombs, secret filenames, and secret content markers.
- Added a scope-aware, non-destructive central refresh helper for Claude and
  Codex. It refuses marketplace source collisions and duplicate Claude scopes.
- Added immutable signed-tag bindings for version, source commit, payload,
  requirements lock, and wheelhouse.
- Bound tag verification to policy and signer bytes from the peeled tag commit,
  enforced the canonical source repository, and rejected tag mutation.
- Added checked-in pre-commit and pre-push blockers. Sensitive pushes require
  an exact-commit full-release stamp and reject policy shrinkage or post-stamp
  tracked drift. Hooks clear Git's repository-local environment before child
  checks so nested temporary repositories cannot mutate the parent checkout.
- Disabled GitHub-hosted Actions for this repository. The former workflow is
  preserved as `.github/workflows-disabled/ci.yml.backup-2026-07-15`.
- Updated current installation and maintainer guidance to distinguish verified
  local acceptance from unverified marketplace UI admission.
